PERJALANAN HIDUP SAYA BELAJAR CODING

Berawal dari Masuk Sekolah Menengah Atas, tetapi masih bingung antara memilih SMA atau SMK karena saya pada saat itu belum ada motivasi untuk menjadi apa, dan ingin seperti apa, karena saya sama sekali tidak mempunyai kompetensi di bidang apapun.

Pada suatu ketika saya akhirnya memutuskan untuk masuk di SMK mengambil jurusan Komputer, pada saat itu masih belum terpikirkan kenapa mengambil jurusan komputer terutama Teknik Komputer Jaringan, karena saya sama sekali gaptek akan komputer. Akhirnya saya memilih jurusan Teknik Komputer Jaringan di salah satu SMK Swasta di Kendal, akan tetapi saya masih belum bisa mengoperasikan komputer dengan baik, bahkan ketika praktek pun saya masih mengandalkan teman untuk membantu saya sepenuhnya karena saya masih takut untuk mengoperasikan komputer karena saya berpikir komputer akan rusak jika saya sembarangan memakai.

Lambat laun berjalannya waktu menjelang akhir semester 1 kelas X, saya sering di ajak teman untuk bermain di warnet setelah sepulang sekolah, melihat teman masuk ke billing warnet dan mengoperasikan software komputer, browsing gambar, artikel dan mendownload sebuah lagu. Akhirnya membuat saya mulai memberanikan diri untuk merental komputer warnet di bilik sendiri, dan akhirnya saya mulai mencobanya, saya mulai mencoba membuka aplikasi web browser mozilla dan mulai browsing mencari gambar gambar anime.

Awal saya menyukai dunia coding adalah ketika saya melihat guru mapel Web Design saya membuat tulisan berjalan menggunakan teks editor notepad, setelah itu saya mulai tertarik dengan dunia coding dan mempunyai rasa penasaran yang tinggi terutama tentang tulisan berjalan itu. Saya yang sebelumnya ketika bermain di warnet hanya sekedar browsing gambar saja mulai mencari tutorial cara membuat tulisan berjalan yang saya sebut sebagai tulisan berjalan di home page. Hampir satu minggu saya mencari cara untuk membuat tulisan berjalan itu dan akhirnya berhasil, padahal hanya sekedar tulisan berjalan memakai syntax marquee, namun itu membuat saya pantang menyerah dan akhirnya saya berhasil.

Setelah mampu membuat tulisan berjalan akhirnya saya mencoba mempelajari tentang HTML dan saya praktekkan hanya dengan menggunakan teks editor notepad karena dulu saya tidak mengenal software Sublime, Visual Studio Code atau yang lainnya. Setelah saya naik kelas XI, ada penawaran untuk mengikuti lomba Kompetensi siswa atau LKS Web Design yang diselenggarakan setiap tahun oleh Kemendikbud, kemudian saya memberanikan diri untuk mengikuti seleksi LKS Web Design untuk mewakili sekolah bertanding di tingkat Kabupaten, dan kemudian saya mulai mengenal software untuk membuat desain web menjadi lebih praktis yaitu Macromedia Dreamweaver walaupun dalam dunia coding, software tersebut tidak direkomendasikan karena tidak bisa membuat seorang web developer berkembang pengetahuan dan keterampilannya tentang coding web, akan tetapi dalam perlombaan diijinkan menggunakan software tersebut walaupun secara dasar pengetahuan tentang html dan atributnya saya sudah menguasainya, dan alhamdulillah mendapatkan juara III meskipun saya bukan dari jurusan Rekayasa Perangkat Lunak yang notabene materinya yang dipelajari tentang coding.
